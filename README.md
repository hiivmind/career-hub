## Quick Start

### What This Skill Does

The **cv-optimization-dual-ai** skill helps job seekers optimize their CVs to pass through two opposing AI systems simultaneously:

1. **ATS Screening** - Employer systems that reject 75% of resumes before human review
2. **Fraud Detection** - Employer verification that identifies fabricated claims with 95% accuracy

### Two-Output Workflow

When a user provides their CV, Claude produces:

1. **Reformatted CV (.docx)** - Ready to submit with:
   - ATS-optimized formatting (single column, standard fonts)
   - Keyword extraction from job description (if provided)
   - Achievement bullets with quantification
   - Fraud-proof claims verified against LinkedIn

2. **Strategy Document (Markdown)** - Explaining:
   - Every change made and why
   - How each change improves ATS parsing
   - How each change reduces fraud red flags
   - Employment gap handling
   - Customization guidance for future jobs

### Key Principles

- **Authentic optimization** - Never encourages fabrication or lies
- **Dual-AI balanced** - Optimizes for both automated screening AND human verification
- **Verifiable claims** - Every achievement bullet can be defended in technical interviews
- **Efficient customization** - 15-20 minute workflow for tailoring to specific jobs

## Philosophy

This skill is built on one fundamental principle: **authentic optimization**.

The goal is never to help anyone lie on their CV. Instead, it helps good candidates present themselves honestly in a way that:
- Passes automated ATS screening
- Survives employer fraud detection
- Demonstrates genuine value in technical interviews
- Enables successful career transitions

In 2025, job applicants face sophisticated AI systems designed to detect fraud. The only sustainable approach is to optimize *authentically*: tell your real story in a way that achieves your career goals.

This skill makes that possible.
